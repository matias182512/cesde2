Seguridad en redes

La seguridad de red es el conjunto de estrategias, procesos y tecnologías diseñados para proteger la red de una empresa frente a daños y al acceso no autorizado. Las amenazas habituales a las que se enfrentan la infraestructura y los datos de una red incluyen hackers, programas maliciosos y virus, todos los cuales pueden intentar acceder a la red, modificarla o dañarla. Una prioridad esencial para la seguridad de la red es controlar el acceso e impedir que estas amenazas entren y se propaguen a través de ella.

aprender a manejar maquinas virtuale

En informática, una máquina virtual es un software que simula un sistema de computación y puede ejecutar programas como si fuese una computadora real. Este software en un principio fue definido como "un duplicado eficiente y aislado de una máquina física".

Herramientas ofimaticas

Bases de datos en exel

es una herramienta para recopilar y organizar información. Las bases de datos pueden almacenar información sobre personas, productos, pedidos u otras cosas. Muchas bases de datos comienzan como una lista en una hoja de cálculo o en un programa de procesamiento de texto.

Sql

SQL es un lenguaje de computación para trabajar con conjuntos de datos y las relaciones entre ellos. Los programas de bases de datos relacionales, como Microsoft Office Access, usan SQL para trabajar con datos. A diferencia de muchos lenguajes de computación, SQL no es difícil de leer y entender, incluso para un usuario inexperto. Al igual que muchos lenguajes de computación, SQL es un estándar internacional reconocido por organismos de estándares como ISO y ANSI.

Power BI

Power BI es un servicio de análisis de datos de Microsoft orientado a proporcionar visualizaciones interactivas y capacidades de inteligencia empresarial con una interfaz lo suficientemente simple como para que los usuarios finales puedan crear por sí mismos sus propios informes y paneles
